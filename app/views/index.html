UR round 1

Changes in this version

This is the first version of the hub design during this new phase of work.

As we kick-off this work, we stripped out the majority of the previous design features (for now), with an aim to carry out some UR to identify the most important features to focus on MVP. We then carried out a card-sorting activity with users, primarily from reception prisons.

We'll use this UR to inform our initial designs for build and to help us target specific features to research more in-depth in the next round.
